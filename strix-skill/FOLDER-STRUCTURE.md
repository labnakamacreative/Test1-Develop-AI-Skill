# Folder Structure: Strix Researcher Skill (Commercial)

```
strix-skill/
├── SYSTEM-PROMPT.md                  # System prompt utama Strix
├── SKILL.md                          # Skill definition + panduan lengkap + mode kerja
├── FOLDER-STRUCTURE.md               # Dokumentasi struktur folder (file ini)
├── standards/
│   ├── _FINAL-AI-humanize-writing-guide.md # ⚠️ WAJIB BACA SELALU, Protokol humanize writing (Tier 1 scan mandatory)
│   ├── standar-riset-audiens.md      # Standar minimum riset audiens
│   ├── standar-riset-kompetitor.md   # Standar minimum riset kompetitor
│   ├── standar-riset-kreator.md      # Standar minimum riset kreator
│   ├── standar-riset-kata-kunci.md   # Standar minimum riset kata kunci
│   ├── standar-riset-trend.md        # Standar minimum riset trend & platform
│   ├── standar-deskripsi-konten.md   # 10 layer WAJIB deskripsi konten
│   ├── standar-penulisan-insight.md  # Cara menulis insight yang tajam
│   ├── standar-penulisan-rekomendasi.md # Standar Level 3 rekomendasi (briefable)
│   ├── standar-evaluasi-konten.md    # Standar evaluasi/audit konten brand
│   └── anti-generik-rules.md        # Red-flag phrases & anti-generik rules
├── frameworks/
│   ├── data-to-insight-framework.md  # Hirarki Data → Informasi → Insight → Decision
│   ├── pattern-recognition.md        # Cara membaca pola dari data riset
│   ├── gap-analysis-framework.md     # Gap analysis & opportunity mapping
│   └── evaluasi-review-checklist.md  # Checklist review/koreksi riset
└── references/
    ├── case-pattern-library.md       # Pola strategis dari studi kasus sosmed
    ├── trend-research-structure.md   # Kerangka struktur riset trend untuk brand
    └── strategy-research-structure.md # Pola & struktur strategy research komprehensif
```

## Cara Baca

1. **SYSTEM-PROMPT.md**. Identitas, prinsip, cara kerja Strix
2. **SKILL.md**. Detail kemampuan, mode kerja, standar output
3. **standards/_FINAL-AI-humanize-writing-guide.md**. ⚠️ WAJIB BACA SELALU di setiap session. Protokol humanize writing yang berlaku di semua output. Jalankan Scan Tier 1 sebelum kirim setiap output.
4. **standards/** (lainnya), WAJIB BACA sebelum mengerjakan jenis riset tertentu
5. **frameworks/**. Kerangka analisis saat dibutuhkan
6. **references/**. Referensi pola dan struktur

## Prinsip Arsitektur

- **Humanize Writing Guide** adalah protokol global yang berlaku di semua output, override file referensi lain dalam hal gaya penulisan
- **Brand Profile** sebagai fondasi konteks semua riset
- **Standards** bersifat universal, applicable untuk semua industri dan brand
- **Frameworks** adalah alat analisis, bukan template rigid
- **References** berisi pola dan struktur, bukan template yang di-copy
